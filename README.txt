To Start Project:

Open:
NTTProject/src/main/resources/application.properties

Edit first 3 lines with the following :

spring.datasource.url=jdbc:mysql://localhost:3306/nttproject?useSSL=false  {Localhost:3306 -> your Mysql Database location; nttproject -> Your Database Name}
spring.datasource.username=root {root -> Username}
spring.datasource.password=0101 {0101 -> Password}
